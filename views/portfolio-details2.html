Adventure Game

  • Category: Python
  • Client: Udacity
  • Project year: 2022
  • Project URL: https://github.com/svolcov1/adventure-game

As you will see, this is only a very short game with a couple of choices available to the player. I did that on purpose, to keep it simple.

The idea is to focus on some key things that we need if want to make a working game such as giving players a description of what's happening, and then asks them to make a choice. In addition, the game includes random factors so it's a little different each time as well as conditions for winning and losing.
